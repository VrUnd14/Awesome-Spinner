# Awesome-Spinner

### Installation
In your project's build.gradle file:
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
In your Application's or Module's build.gradle file:
```
dependencies {
    implementation 'com.github.VrUnd14:Awesome-Spinner:1.0.0'
}
```
